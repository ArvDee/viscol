Example: Loading multiple files

There are different methods to load multiple configurations into a single viewer. The easiest way is to supply a list of multiple snapshot urls instead of a single url. 


  var viewer=new viscol(document.getElementById("webgl-canvas"),"static");
  viewer.loadFilesFromServer([
    'snaps/snap_0.dat',
    'snaps/snap_1.dat',
    'snaps/snap_2.dat',
    'snaps/snap_3.dat',
    'snaps/snap_4.dat',
    'snaps/snap_5.dat',
    'snaps/snap_6.dat',
    'snaps/snap_7.dat',
    'snaps/snap_8.dat',
    'snaps/snap_9.dat']);
  viewer.toggleFrameTitle(false);
  viewer.toggleDrawBox(false);
  viewer.setZoom(1.2);
  viewer.setSpeed([1,1]);
  viewer.togglePlay();
  viewer.setPlayDelay(10);
The number of frames (the visualization aims to run at 60 fps) between configuration switches is set by the setPlayDelay function. Instead of automatically cycling between the frames it is possible to use html buttons to switch between configurations.
