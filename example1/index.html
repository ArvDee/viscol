Example: Simple static viewer

It is possible to display a non interactive view of by creating a canvas and passing the canvas to the viewer. The viewer can then be used to load a configuration file. In this case from the local path on the server.

You need to load the javascript libraries in the html file: 


<script type="text/javascript" src="gl-matrix.js"> </script>
<script type="text/javascript" src="twgl-full.min.js"> </script>
<script type="text/javascript" src="viscol.js"> </script>
And you need to create a canvas that will be used to display the visualization: 

<body onload="start();">
<canvas id="webgl-canvas" width="300" height="300"> </canvas>
This javascript code needs to run when the web page is loaded so we attach it to the onload event of the body. To load a snapshot we place one in snaps/snap_1.dat and load it once the viewer is created. We then set a fixed rotation speed to provide a changing view of the snapshot. 

<script type="text/javascript">
function start(){
  var viewer=new viscol(document.getElementById("webgl-canvas")); 
  viewer.loadFilesFromServer('snap_1.dat');
  viewer.toggleDrawBox(false);
  viewer.setZoom(1.2);
  viewer.setSpeed([1,1]);
}
</script>
