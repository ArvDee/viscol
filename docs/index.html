Introduction

Viscol is an online tool to visualize colloidal particles in 3D in a browser. It can be used to view, inspect and manipulate 3D configurations of particles. Different particles shapes can be displayed such as spheres, cubes and rods and it is easily extendible to other particle shapes.

Documentation

The project is split over 3 objects. The core object viscol_core contains the core functions used to display frames and particles. The second object viscol is used to handle user input and to load frames and particles into the core object. The third object is only used to provide different particle shapes that can be loaded into the core object and can be used to load particle shapes from wavefront .obj files.
