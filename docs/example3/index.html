Example: Configuration file format for spheres

The most basic snapshot file contains spheres the number of paricles on the first line optionally followed by a title string. The next three lines contain the dimensions of the box. For spheres the format is as follows: 


Number_of_particles [A title string]
-xbox xbox
-ybox ybox
-zbox zbox
x1 y1 z1 diam1 [color1]
x2 y2 z2 diam2 [color2]
...
where the color is optional and can be either a single integer or 3 rgb values.

For example for a red and a green sphere this becomes: 


2 Two touching spheres
-1.2  1.2
-1.2  1.2
-1.2  1.2
 0.5  0.0  0.0 1.0 1
-0.5  0.0  0.0 1.0 2
