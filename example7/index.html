Example: Saving png files

If you want to save a png file it is possible to do so by calling the capturePng function of the viewer.






In html we only need to add a button to save a frame of a movie: 


<button onclick="viewer.capturePng()">save PNG </button>
<button onclick="viewer.captureMovie(20)">Save movie</button>
20 is the number of frames that is captured.

For the movie functionality to work we also need to load the CCapture java script library: 


<script type="text/javascript" src="../external/CCapture.all.min.js"></script>
